# CreditRiskPredict - прогнозирование кредитного риска

## Цель — построить модель, которая оценивает вероятность дефолта по кредиту, помогая минимизировать финансовые потери

## Основные возможности

- **Обработка и EDA** (`data_EDA`): исследование данных, очистка, визуализация
- **Обучение моделей** (`models.py`): реализация нескольких алгоритмов классификации (логистическая регрессия, дерево решений, случайный лес)
- **Оценка качества**: метрики accuracy, precision, recall, F1, ROC-AUC
- **Интерпретация**: важность признаков, отчеты по классификации

## Структура проекта

- data_EDA: Первичное исследование датасета и зависимостей, обнаружение пропусков и выбросов (`raw.py`); Подготовка данных для дальнейшего обучения моделей (`processed.py`)
- models.py: Обучение и оценка моделей, визуализация результатов
- requirements.txt: Список используемых библиотек

## Установка и запуск

1. Установите датасет:
   https://www.kaggle.com/datasets/laotse/credit-risk-dataset/data
   
   Добавьте датасет в папку `data_EDA`
   
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Обучите модель:
   ```bash
   python models.py
   ```

При необходимости:
  - Запустите `raw.py` чтобы увидеть информацию об изначальном датасете
    ```bash
    python data_EDA\raw.py
    ```

## Используемые технологии

- Python 3.12
- pandas, numpy – обработка данных
- matplotlib, seaborn – визуализация
- scikit-learn – модели машинного обучения
